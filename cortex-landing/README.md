# Cortex Landing Page

Standalone Next.js marketing site for Cortex. It is intentionally independent from the existing `ui/` SolidJS application and contains no backend or product API connections.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

The alpha CTA currently opens an email draft to `hello@cortex.dev` so the page remains backend-free.
